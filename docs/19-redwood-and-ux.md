# Redwood and UX

## UX Perspective

The user-facing experience is the visible tip of a much larger configuration model.

## Modern Experience Themes

- Enhanced / unified time cards
- Redwood-aligned experiences
- consolidated layouts for multiple roles

## What Good UX Depends On

- correct entry profile assignment
- clean layout design
- sensible field exposure
- limited unnecessary prompts

## UX Guidance for an Interactive Learning Site

- introduce plain English first, then the Oracle term
- show object relationships visually
- separate setup, runtime, and downstream views
- use progressive disclosure for complex objects
