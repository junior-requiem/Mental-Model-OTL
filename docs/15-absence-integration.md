# Absence Integration

## Purpose

Absence integration allows leave-related time to participate in the broader time truth maintained by OTL.

## How It Differs

Absence can act like a parallel consumer path rather than a classic transfer-only downstream pattern.

## What It Relies On

- Absence-related time attributes
- Entry design that exposes absence reporting when intended
- Rules that validate balance and eligibility context

## Common Mistakes

- Treating absence like regular work time
- Missing absence attributes or balances
- Showing absence UI to the wrong population
