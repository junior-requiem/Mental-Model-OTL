# Web Clock and Time Devices

## Web Clock

Web Clock is Oracle's event-style time capture experience for recording in/out events.

## Time Device Processing

Third-party clocks and devices require a translation and processing layer.

### Main Objects

- Time Device Event Mapping
- Time Device Event Mapping Set
- Time Device Rule Set
- Submission Rule / Time Submission Rule Set
- Time Device Processing Profile

## Mental Model

Web Clock is the native event capture path. Time device processing is the bridge for external event sources.

## Common Mistakes

- Wrong event mapping
- Using mapping sets for Web Clock scenarios
- Missing Time Attributes on Web Clock buttons
- Missing device processing profile assignment
