# Layout Sets and Components

## What They Are

Layouts define the assembled user experience for time entry.

- Layout Components are the reusable building blocks.
- Layout Sets are the assembled experience delivered to a population.

## What They Control

- Which fields appear
- Which actions and buttons appear
- How time cards, calendars, dialogs, and Web Clock pages behave

## Mental Model

Layout Components are the pieces on the screen. Layout Sets are the assembled screen architecture.

## Common Mistakes

- Building too many unique layouts
- Missing required fields
- Exposing conflicting fields for the same data concept
- Overcomplicating unified experiences with too many dependencies
