# Payroll Integration

## Purpose

Payroll integration turns governed OTL time into payable time for compensation processing.

## What Payroll Needs

- Valid time
- Correct payroll-related attributes
- Correct status and approval state
- Proper mapping to payroll elements or related structures

## Why It Is Not Just a Connector

Payroll integration depends on entry design, attributes, rule outcomes, consumer routing, repeating periods, approval, and transfer behavior.

## Common Mistakes

- Misaligned periods
- Missing payroll attributes
- Wrong consumer set
- Investigating payroll only when the issue began earlier in OTL
