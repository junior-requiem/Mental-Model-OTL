# Security and Audit

## Security Model

OTL security determines who can see, enter, edit, approve, and diagnose time.

## Typical Security Building Blocks

- Job Roles
- Data Roles
- Security Profiles
- Population-based access controls

## Audit

Audit records who changed what and why. It provides traceability for time adjustments.

## Common Mistakes

- No required reason codes for adjustments
- Weak manager security
- Missing security setup for analysis tools
