# Project Costing Integration

## Purpose

Project Costing integration enables labor to be charged, costed, and analyzed against projects and tasks.

## What It Depends On

- Project-related attributes
- Valid task and project relationships
- Consumer routing to the correct downstream path
- Correct rule and category behavior

## Important Concepts

- Expenditure organizations
- Transaction controls
- Import or pull-based costing processes

## Common Mistakes

- Project-related fields shown but not properly validated
- Resource is not allowed to charge the task
- Wrong consumer or category mapping
