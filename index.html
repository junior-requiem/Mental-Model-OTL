Expandable Learning Layers

Setup (configuration before daily use)

In this layer you define who gets which policy pack: HCM Groups, Worker Time Entry Profiles, Worker Time Processing Profiles, layout sets, time attributes, categories, and rule sets.

Runtime (daily entry and approval)

Workers and managers capture time through time cards, calendar, web clock, or device imports. OTL then validates entries, derives calculated time, and routes approvals.

Downstream (transfer and consumption)

After required approvals, time reaches ready-to-transfer status and is handed off to payroll, project costing, project execution, extracts, or other external consumers.
